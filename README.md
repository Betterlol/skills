# Skills

本仓库汇集了面向 AI 编码智能体（Claude、Codex、Gemini、OpenCode 等）的 **技能包（Skills）**，用于在工程开发中提供规范化、可执行的指导流程。

## 技能列表

### [Git Stage Manager](./git-stage-manager/README_zh.md)

面向 AI 智能体的版本控制协议。定义了一套严谨的状态机驱动流程，让 AI 编码智能体能够安全、可追溯地完成单阶段 Git 提交。

- **核心能力**：阶段隔离、FILE_SET 显式声明、证据驱动摘要、用户变更保护
- **状态机**：10 状态（IDLE → COMPLETED + RECOVERY）
- **适用场景**：任何需要 AI 智能体进行可控 Git 提价的工程阶段
